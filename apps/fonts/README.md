## Example

> if some font it is not in the example, just update in your css the necessary file endpoint using fonts.azion.com domain;


## Azion Brand Fonts

### Sora

``` css
@font-face {
    font-family: 'Sora';
    src: url('https://fonts.azion.com/sora/Sora-VariableFont_wght.ttf') format('truetype');
    font-weight: 200 900;
    font-style: normal;
    font-display: swap;
}
```
