# Last-modified avatars

Mock "last modified by" photos for the Applications table (see
`src/components/Applications.vue` → `APP_AUTHORS`). Vite serves this folder at
the site root, so each file resolves at `/avatars/<name>.jpg`.

Save each person's photo here with the exact filename below (JPG). Until a file
exists, the `Avatar` falls back to the person's initials — the table still works.
